HOW TO USE THESE FILES IN XCODE
--------------------------------

1. Create a new iOS App project in Xcode:
   - Open Xcode
   - File > New > Project
   - Choose "iOS App"
   - Product Name: AICoach
   - Interface: SwiftUI
   - Language: Swift
   - Click Create

2. Copy all .swift files from this Replit to your Xcode project:
   - Drag and drop all files into your Xcode project navigator
   - Maintain the folder structure

3. Add Dependencies:
   - In Xcode, go to File > Add Package Dependencies
   - Add each dependency from Package.swift:
     * https://github.com/firebase/firebase-ios-sdk.git
     * https://github.com/google/GoogleSignIn-iOS
     * https://github.com/airbnb/lottie-ios.git
     * https://github.com/danielgindi/Charts.git

4. Configure Firebase:
   - Go to https://console.firebase.google.com
   - Create a new project
   - Add an iOS app
   - Download GoogleService-Info.plist
   - Add it to your Xcode project

5. Enable Sign in with Apple:
   - In Xcode, select your project target
   - Go to "Signing & Capabilities"
   - Click "+ Capability"
   - Add "Sign in with Apple"

6. Configure Google Sign-In:
   - Add your Google Sign-In client ID to Info.plist
   - Add URL schemes for Google Sign-In

7. Build and Run:
   - Select your target device or simulator
   - Press Cmd+R to build and run
